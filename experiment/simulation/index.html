📚 Simulation Steps

  1. Open the SBA Simulator Dashboard and verify all panels are visible (NF List, Configuration Panel, Logs, Terminal).
  2. Click Terminal and run docker compose -f docker-compose.yml up -d to start all core network containers.
  3. Run docker compose -f docker-compose-gnb.yml up -d to deploy the gNB and connect it to the core.
  4. Run watch docker compose -f docker-compose.yml ps -a to monitor container status in real time.
  5. From the NF List, click an NF (e.g. NRF first, then AMF, SMF, UPF, etc.) to open its Configuration Panel.
  6. Enter a valid IP address, port number, and HTTP protocol, then click Start NF.
  7. Wait 4–5 seconds for the NF to stabilize and register with the NRF.
  8. Check the Logs Section for "NF started successfully" and "Service registration complete".
  9. Repeat steps 5–8 for all remaining NFs (AMF, SMF, UPF, UDM, AUSF, PCF, etc.).
  10. Select the gNB tile, fill in IP and port, then click Start gNB and wait for it to stabilize.
  11. Click Trigger NGAP Flow in the gNB Configuration Panel to establish the control plane link with AMF.
  12. Click Trigger GTP-U Packet to activate the user plane tunnel between gNB and UPF.
  13. Click an NF, open its Command Prompt, and run ping <target_IP> to validate NF-to-NF connectivity.
  14. Check ping output — zero packet loss confirms the components can reach each other.
